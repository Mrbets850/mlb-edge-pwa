# MLB EDGE PWA Installer

This repository hosts the standalone GitHub Pages installer for MLB EDGE.

Live app:

https://mrbets850.streamlit.app/

Expected GitHub Pages URL after publishing:

https://mrbets850.github.io/mlb-edge-pwa/

## Customer install instructions

### iPhone

1. Open the GitHub Pages URL in Safari.
2. Tap Share.
3. Tap Add to Home Screen.
4. Tap Add.

### Android

1. Open the GitHub Pages URL in Chrome.
2. Tap Install MLB EDGE, or use the three-dot menu and tap Install app.
3. The gold and purple MLB EDGE icon appears on the Home Screen.

### Discord

Discord's in-app browser cannot install PWAs. Open the link in Safari or Chrome first.
